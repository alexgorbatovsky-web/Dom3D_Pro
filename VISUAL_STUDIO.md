# Работа в Visual Studio

Проект уже подготовлен как обычное решение Visual Studio.

## Как открыть

Открой файл:

```text
C:\My_projects\Dom3D_Pro\build\Dom3D_Pro.sln
```

Внутри решения главный проект:

```text
Dom3D_Pro
```

## Как запустить

1. В Visual Studio выбери конфигурацию `Release` или `Debug`.
2. В `Solution Explorer` нажми правой кнопкой на `Dom3D_Pro`.
3. Выбери `Set as Startup Project`.
4. Нажми `F5` для запуска с отладкой или `Ctrl+F5` для обычного запуска.

## Где исходники

Исходный код лежит здесь:

```text
C:\My_projects\Dom3D_Pro\src
```

Основные классы:

- `Dom3DApp` — приложение, окно, меню, команды, мышь
- `CAlfaObject` — базовый объект сцены с виртуальными `Render3d`, `Render2d`, `HitTest`, `Save`, `Edit`
- `CAlfaDoc` — документ и список объектов `CAlfaObject`
- `CPolyline` — полилиния с точками, именем, цветом, сохранением, выбором и собственным рендером
- `CView3d` — 3D-сцена, обходит объекты документа и вызывает `Render3d`
- `CView2d` — 2D-чертежное представление, обходит объекты документа и вызывает `Render2d`
- `Renderer` — OpenGL-контекст и общий рендеринг
- `ProjectIO` — сохранение и загрузка `.d3dm`

Поведение инструмента `Curve`:

- каждое нажатие `Curve` создает новую активную `CPolyline`;
- `Esc` завершает создание кривой и возвращает режим `Orbit`.
- `Select` выбирает объект в сцене;
- в режиме `Select` можно выбрать и перетащить точку кривой;
- `Delete` удаляет выбранную точку, а если точка не выбрана — выбранный объект;
- двойной клик по объекту вызывает его виртуальный метод `Edit`.
- средняя кнопка мыши/колесико двигает сцену.

## Если добавляем новые файлы

Новые `.cpp` файлы нужно добавить в:

```text
C:\My_projects\Dom3D_Pro\CMakeLists.txt
```

После этого Visual Studio обычно сама обновит проект. Если не обновит, можно открыть папку проекта через:

```text
File -> Open -> Folder...
```

и выбрать:

```text
C:\My_projects\Dom3D_Pro
```
