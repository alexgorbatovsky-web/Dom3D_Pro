# Dom3D Pro

Прототип современного десктопного приложения на C++ с OpenGL.

## Что уже есть

- Обычное Windows-окно
- Главное меню `File`, `Tools`, `Curve`
- Верхняя панель инструментов
- Архитектура через C++ классы:
  - `Dom3DApp` — приложение, окно, меню, команды и мышь
  - `CAlfaObject` — базовый объект сцены с виртуальными `Render3d`, `Render2d`, `HitTest`, `Save`, `Edit`
  - `CAlfaDoc` — документ и список объектов `CAlfaObject`
  - `CPolyline` — полилиния с точками, именем, цветом, сохранением, выбором и рендером
  - `CView3d` — 3D-представление документа, обходит объекты и вызывает их `Render3d`
  - `CView2d` — 2D-представление документа, обходит объекты и вызывает их `Render2d`
  - `Renderer` — OpenGL-контекст и отрисовка
  - `ProjectIO` — сохранение и загрузка `.d3dm`
- OpenGL-сцена с перспективной камерой
- Простая 3D-комната с полом, стенами и объектами
- Темная боковая панель интерфейса
- Вращение камеры мышью
- Масштаб колесиком мыши
- Моделирование кривой кликами мыши по полу
- Выбор объектов через инструмент `Select`
- Удаление выбранного объекта клавишей `Delete`
- Двойной клик по объекту вызывает его виртуальный метод `Edit`
- Сохранение и загрузка проекта в формате `.d3dm`
- Формат проекта хранит набор полилиний, а не одну временную кривую

## Работа с кривой

1. Нажми `Curve` на верхней панели или выбери `Tools -> Draw curve` — создастся новая кривая.
2. Кликни несколько точек на полу сцены.
3. Нажми `Save` или выбери `File -> Save...`.
4. Открой сохраненный файл через `Open` или `File -> Open...`.

Для вращения сцены нажми `Orbit` или выбери `Tools -> Orbit camera`.
Клавиша `Esc` завершает создание кривой и возвращает режим `Orbit`.
Средняя кнопка мыши/колесико сдвигает сцену.

## Выбор и редактирование объекта

1. Нажми `Select`.
2. Кликни по кривой в сцене.
3. Кликни по точке кривой и перетащи ее мышью.
4. Нажми `Delete`, чтобы удалить выбранную точку; если точка не выбрана, удалится выбранная кривая.
5. Сделай двойной клик по кривой, чтобы вызвать `CPolyline::Edit`.

## Сборка

### Через Visual Studio

Открой готовое решение:

```text
C:\My_projects\Dom3D_Pro\build\Dom3D_Pro.sln
```

Подробно: `VISUAL_STUDIO.md`.

### Через консоль

Открой `Developer Command Prompt for Visual Studio` и выполни:

```bat
cd C:\My_projects\Dom3D_Pro
cmake -S . -B build
cmake --build build --config Release
```

Готовый файл:

```text
C:\My_projects\Dom3D_Pro\build\Release\Dom3D_Pro.exe
```
